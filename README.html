Traffic Light Optimization Simulation

Описание проекта

Это симулятор оптимизации светофоров на перекрестке, написанный на Python с использованием библиотеки Pygame. Программа визуализирует движение транспортных средств и работу светофоров, позволяя регулировать плотность потока машин в разных направлениях.

Основные функции

  • Реалистичная визуализация перекрестка с двумя светофорами
  • Движение машин с соблюдением правил дорожного движения (остановка на красный свет)
  • Возможность регулировки плотности транспортного потока
  • Автоматическое переключение светофоров
  • Приветственный экран и интуитивно понятный интерфейс

Требования

  • Python 3.6 или новее
  • Библиотека Pygame

Установка

  1. Убедитесь, что у вас установлен Python 3.x
  2. Установите необходимые зависимости: 
    pip install pygame

Запуск программы

python traffic_simulation.py

Использование

  1. Нажмите "Start" на приветственном экране
  2. Используйте ползунки для регулировки плотности потока машин:
    • Левый ползунок - транспорт слева направо
    • Правый ползунок - транспорт справа налево
  3. Наблюдайте за работой светофоров и поведением машин
  4. Для выхода нажмите кнопку "Exit"

Особенности реализации

  • Класс Car: отвечает за поведение и отрисовку отдельных машин
  • Светофоры: работают в противофазе (когда один зеленый, другой красный)
  • Физика движения: машины плавно останавливаются на красный свет
  • Интерфейс: включает индикаторы состояния светофоров и плотности потока

Структура кода

traffic_simulation.py ├── Константы (размеры окна, цвета, настройки) ├── Класс Car (логика движения машин) ├── Функции отрисовки: │ ├── draw_traffic_light │ ├── draw_cars │ └── draw_interface ├── Вспомогательные функции: │ ├── initialize_pygame │ └── load_font └── Основная функция main()
